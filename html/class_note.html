Detailed Description

Définit la classe Notes : Permet de d'ajouter, supprimer, restaurer une Version à une Note.

Design Pattern : Iterator. id : identifiant de la note : unique version : tableau de pointeur de Version nbVersion : nombre de versions de la note nbMaxVersion : nombre de versions maximales de la note etat : Etat de la note date_creation : Date de création de la note
